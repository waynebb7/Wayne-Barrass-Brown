Image Manipulation

Borders, Padding & Spacing

Border styles:
none
hidden
dotted
dashed
solid
double
groove
ridge
inset
outset

Setting Margins (space inside the border)

Margins can be set:
margin-left: length
margin-right: length
margin-top: length
margin-bottom: length
margin:top right bottom left;
margin: top-and-bottom right-and-left

Setting Padding (space outside the border)

Padding can be set:
padding-left: length
padding-right: length
padding-top: length
padding-bottom: length
padding:top right bottom left;
padding: top-and-bottom right-and-left

Text - Vertical Alignment

Vertical alignment can be set:
vertical-align: baseline
vertical-align: sub
vertical-align: super
vertical-align: text-top
vertical-align: text-bottom
vertical-align: middle
vertical-align: top
vertical-align: bottom
vertical-align: percentage
vertical-align: length

Text flow around image

Float can be set:
float: left
float: right
float: inline-start
float: inline-end

Example: style="float:left; padding: 0 0.5em 0.5em 0"

Example: style="clear: left" This prevents text starting until it is vertically clear of the image on the LEFT. (right & both can be used)

Rotate an image

Rotation can be set:
style="transform: rotate(-20deg)

Move an image in X or Y direction using translate

TranslationX can be set:
style="translateX: (30px)

TranslationY can be set:
style="translateY: (30px)

Modify image to fit inside a shape

clip-path can be set:
style="clip-path: circle(radius at x, y)

clip-path can be set:
style="clip-path: ellipse(radius radius at x, y)

clip-path can be set:
style="clip-path: polygon(x1, y1, x2, y2, ...)

Modify image to be a fixed background

style="background:
url("xxxxxxx.png")
no-repeat
right 20px top 20 px
fixed;

Responsive Images

This code is always required: <meta name="viewport" content="width=device-width, initial-scale=1.0">

Plus this in the image section: srcset="image1.jpg, image2.jpg, ..."

Use sizes to ensure that each picture displays at 50% viewport width. sizes="50vw"

Refer to this course 'Using srcset and sizes' as the demo is good.
